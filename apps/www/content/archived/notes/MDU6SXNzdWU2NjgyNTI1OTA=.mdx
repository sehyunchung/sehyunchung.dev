1. asdf를 설치한다.
   ```sh
   brew install asdf
   ```
1. gpg를 설치한다.
   ```sh
   brew install gpg
   ```
1. asdf nodejs plugin을 설치한다.
   ```sh
   asdf plugin-add nodejs https://github.com/asdf-vm/asdf-nodejs.git
   ```
1. nodejs release team의 gpg key를 추가한다.
   ```sh
   bash -c '${ASDF_DATA_DIR:=$HOME/.asdf}/plugins/nodejs/bin/import-release-team-keyring'
   ```
1. `nvm`등 사용중이던 vm이 있다면 호환성을 위해 `~/.asdfrc`에 아래를 추가한다.
   ```sh
   legacy_version_file = yes
   ```
1. 됫다.
