js array는 object이고 `length` 프로퍼티를 갖고 있다. 그래서 아래가 가능.

```ts
const {length, [0]: first, [length - 1]: last} = [1, 2, 3];

console.log(first, last);  // 1 3
```

주의: `length`를 선언해두지 않으면 `length - 1`을 알 수 없고 `last`가 `undefined` 되어버림.
